# claude-practice

Claude Codeの練習用リポジトリです。

## 目的

- Claude Codeの使い方を学ぶ
- GitとGitHubの基本的なワークフローを練習する
- PRの作成・レビューを体験する

## 使い方

このリポジトリはClaude Codeと一緒に学習するための練習場所です。

## はじめかた

1. このリポジトリをクローンする
   ```bash
   git clone https://github.com/xyoumo-kgc/claude-practice.git
   ```
2. ディレクトリに移動する
   ```bash
   cd claude-practice
   ```
3. Claude Codeを起動して練習を始める
   ```bash
   claude
   ```
