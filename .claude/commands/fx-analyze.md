# FXマルチタイムフレーム分析スキル

## 概要

このスキルは「4pattern」手法に基づくFXのマルチタイムフレーム分析を行います。
3つの時間軸（上位足・中間足・下位足）の関係から4つのパターンを判定し、
トレード可否と方向性を提示します。

## 使い方

```
/fx-analyze <通貨ペア> <日足トレンド> <4時間足トレンド> <1時間足トレンド>
```

例: `/fx-analyze USDJPY 上昇 上昇 下降`

引数省略時はユーザーに質問します。

---

## スキル実行手順

ユーザーから通貨ペアと各時間軸のトレンド情報を受け取り、以下の手順で分析してください。

### ステップ1: 情報収集

引数が不足している場合は以下を質問してください：

1. **通貨ペア**（例: USDJPY, EURUSD）
2. **日足のトレンド**（上昇 / 下降 / レンジ）
3. **4時間足のトレンド**（上昇 / 下降 / レンジ）
4. **1時間足のトレンド**（上昇 / 下降 / レンジ）
5. **日足MA(120期間)と価格の位置関係**（MA上 / MA下 / MA付近）
6. **4時間足MA(20期間)と1時間足MA(80期間)の収束・拡散状態**（収束中 / 拡散中）

### ステップ2: 3つの流れを確認

各時間軸のトレンドを「大きな流れ」「中くらいの流れ」「小さな流れ」として整理：

| 流れ | 時間軸 | MAの目安 |
|------|--------|---------|
| 大きな流れ（プライマリー） | 日足 | 日足MA(120〜480期間) |
| 中くらいの流れ | 4時間足 | 4H MA(80期間) |
| 小さな流れ | 1時間足 | 1H MA(20期間) |

### ステップ3: 4パターン判定

以下の4パターンに当てはめて判定してください：

---

#### 🟢 パターン1 — 全方向一致・最高優先度

**条件:** 日足 ↑ ＋ 4時間足 ↑ ＋ 1時間足 ↑ （または 日足 ↓ ＋ 4時間足 ↓ ＋ 1時間足 ↓）

**判定:** トレードOK（最も高確率）

**解説:**
- 3つの時間軸がすべて同方向に向いているため、トレードの優位性が最も高い
- 上昇方向なら「押し目買い」を狙う
- 下降方向なら「戻り売り」を狙う
- MAが拡散していれば上位足の勝ちが確定している状態

**エントリー条件（買いの場合）:**
- 1時間足の安値切り上げポイントでロング
- 4時間足レベルの押し安値付近が最良のエントリーゾーン
- SLは直近の押し安値の下

---

#### 🟡 パターン2 — 上位2つ一致・下位足が逆行中

**条件:** 日足 ↑ ＋ 4時間足 ↑ ＋ 1時間足 ↓（または 日足 ↓ ＋ 4時間足 ↓ ＋ 1時間足 ↑）

**判定:** 押し目（戻り目）形成中 → 方向性一致後にトレードOK

**解説:**
- 日足・4時間足は上昇トレンド中だが、1時間足が一時的に逆行（押し目を形成中）
- プライマリートレンドに逆行した動きは一時的で、必ずプライマリー方向に回帰する
- 1時間足が上昇トレンドに転換した時点が「4時間足レベルの押し目買いチャンス」

**エントリー条件（買い方向の場合）:**
- 1時間足で「安値切り上げ → 上昇トレンド転換確認」後にロング
- 4時間足レベルの押し安値圏内での1時間足転換が高確率
- 1時間足が上昇転換する前に買うのはNG（「まだ下降トレンド中」の場合は待つ）

---

#### 🔴 パターン3 — 上位足 vs 中間足が対立（戦争中）

**条件:** 日足 ↑ ＋ 4時間足 ↓（または 日足 ↓ ＋ 4時間足 ↑）

**判定:** 難易度高・基本はトレードNG or 極小ロット

**解説:**
- 「戦争中（上位足の勝ちがまだ確定していない）」の状態
- 日足レベルのトレーダーは買いたいが、4時間足レベルのトレーダーは売りたい
- MAが収束しているほど「戦争中」→ MAが拡散し始めたら上位足の勝ち確定
- **MA収束時:** 上位足と下位足の攻防中
- **MA拡散時:** 上位足の勝ちが確定

**この場面でのNG行動:**
- 日足上昇だからといって安易に買わない
- 4時間足下降だからといって売りを仕掛けない
- 戻り高値を超えていない＋安値切り上げていない段階でのエントリーはNG

---

#### 🟠 パターン4 — 上位足方向へのカウンター（逆張り禁止）

**条件:** 日足 ↓ ＋ 4時間足 ↓ ＋ 1時間足 ↑（日足・4時間足に逆らう小さな反発）

**判定:** トレードNG（売り目線を維持）

**解説:**
- 日足・4時間足ともに下降トレンド中の「一時的な戻し（セカンダリートレンド）」
- この上昇は「上位足の下降トレンド継続における戻り目形成」に過ぎない
- 必ずプライマリートレンド（下降）方向に戻ってくる
- 1時間足の上昇に乗って逆張り買いをするのは非常に危険

**正しい行動:**
- 1時間足の上昇が「戻り目」を形成したら、そこからの売りを待つ
- 4時間足レベルの戻り高値付近での1時間足の下降転換が売りのチャンス

---

### ステップ4: ダウ理論によるトレンド確認

各時間軸のトレンドを確認する際は必ずダウ理論で判断してください：

**上昇トレンドの定義:**
- 高値が切り上がり（前の高値を更新）かつ
- 安値が切り上がり（前の安値を割らない）

**下降トレンドの定義:**
- 高値が切り下がり（前の高値を更新できない）かつ  
- 安値が切り下がり（前の安値を割る）

**トレンド転換の確認:**
- **上昇→下降転換:** 高値切り下げポイントで判断（戻り高値を更新できず、前の安値を割った時点で確定）
- **下降→上昇転換:** 安値切り上げポイントで判断（押し安値を割らず、前の高値を更新した時点で確定）

### ステップ5: エントリー・SL・TP設定ガイド

**エントリーポイントの優先順位:**

1. 最優先: 4時間足レベルの押し目（戻り目）圏内での1時間足転換
2. 次優先: 日足レベルの重要サポート/レジスタンス付近での1時間足転換
3. 注意: 強いサポートライン付近での買いは「売り圧力」が集中しているため逆張りに注意

**決済ライン（SL）の考え方:**
- SLは直近の押し安値（買いの場合）/ 戻り高値（売りの場合）の下/上に設置
- トレンドが進むにつれてSLラインを引き上げ（引き下げ）て利益を保護
- 決済ラインは上昇波の都度引き上げていく（トレーリングストップ的に管理）

**利食いの目安:**
- 日足レベルの次の抵抗/支持帯
- 4時間足MA(20期間)との乖離が大きくなった時
- 1〜4から買っていたトレーダーの決済が集中するポイント付近

## 重要な原則（必ず守ること）

1. **マルチタイムフレーム分析が大前提** — 1つの時間軸だけでトレードしない
2. **上位足の方向に逆らわない** — セカンダリートレンドへの逆張りは厳禁
3. **戦争中（MA収束時）はトレードを控える** — 不確実性が高い
4. **1通貨ペアに集中** — 複数ペアを同時に見るより1ペアを深く分析する
5. **押し安値・戻り高値を常に確認** — トレンド転換の判断基準

---

## 用語集

| 用語 | 説明 |
|------|------|
| プライマリートレンド | 上位足（日足）が示す大きな相場の方向性 |
| セカンダリートレンド | プライマリートレンドに対して一時的に逆行する中間的な動き |
| 押し目 | 上昇トレンド中の一時的な下落（買いのチャンス） |
| 戻り目 | 下降トレンド中の一時的な上昇（売りのチャンス） |
| 押し安値 | 直近の上昇波を作った時の安値（これを割ると上昇トレンド終了） |
| 戻り高値 | 直近の下降波を作った時の高値（これを超えると下降トレンド終了） |
| 上位足の勝ち | MAsが拡散し、上位足の方向性が確定した状態 |
| 戦争中 | MAが収束しており、上位足と下位足のどちらが勝つか未確定な状態 |
| トレードOK | 相場の大きな流れと同方向でエントリーできる状態 |
| トレードNG | 上位足の方向に逆らうためエントリーを控えるべき状態 |
