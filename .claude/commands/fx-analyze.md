# FX 4つの鉄板エントリーパターン分析スキル

## 概要

「100億円トレーダー」による「4つの鉄板エントリーパターン解説マニュアル」に基づく
FXマルチタイムフレーム分析を行うスキルです。

## 使い方

```
/fx-analyze
```

引数なしで起動すると、対話形式で分析に必要な情報を収集します。

---

## スキル実行手順

### ステップ1: 情報収集

以下の質問をユーザーに行い、情報を収集してください：

1. **通貨ペア**（例: USDJPY, EURUSD, GBPUSD など）
2. **日足のトレンド**（上昇 / 下降 / レンジ）
   - ダウ理論で判断：安値切り上げ＋高値更新 → 上昇、高値切り下げ＋安値更新 → 下降
3. **4時間足のトレンド**（上昇 / 下降 / レンジ）
4. **1時間足のトレンド**（上昇 / 下降 / レンジ）
5. **直上にレジスタンスライン・上位足MAなど反対勢力はあるか？**（買いの場合）
   または **直下にサポートラインなど反対勢力はあるか？**（売りの場合）

---

### ステップ2: 基本原則の確認

すべてのパターンに共通する大原則：

> **「大きな流れ（長期・中期の流れ）を見極めて、その中で小さな流れ（トレード足）が大きな流れに合致する瞬間を見つければ良い」**

- **相場は大きな流れに逆らえない** → 大きなトレンドは継続する
- **上位足トレーダーと下位足トレーダーの思惑が一致した局面のみでトレード**する
- 思惑がぶつかり合う「戦争状態」では一切トレードしない

---

### ステップ3: 移動平均線の設定値

各時間軸のMAは、すべての時間軸で「20SMA相当」を統一的に使用：

| チャート | 青線（短期MA） | 赤線（中期MA） | 黄線（長期MA） |
|---------|------------|------------|------------|
| 1時間足 | 20SMA | 80SMA（4H足の20SMA相当） | 480SMA（日足の20SMA相当） |
| 4時間足 | 20SMA | 120SMA（日足の20SMA相当） | 600SMA（週足の20SMA相当） |
| 日足 | 20SMA | 100SMA（週足の20SMA相当） | 400SMA（月足の20SMA相当） |
| 15分足 | 20SMA | 80SMA（1H足の20SMA相当） | 320SMA（4H足の20SMA相当） |

---

### ステップ4: ダウ理論によるトレンド判定

**上昇トレンド（継続）:**
- 安値を切り上げながら高値を更新し続けている状態
- 明確な転換シグナルが出るまで継続とみなす

**下降トレンド（継続）:**
- 高値を切り下げながら安値を更新し続けている状態

**トレンド転換（上昇→下降）:**
- 高値を切り下げ（＝上昇トレンド終了）＋ 安値を更新 → 下降トレンド転換確定
- ダブルトップ形成＋安値更新でも転換とみなせる

**トレンド転換（下降→上昇）:**
- 安値を切り上げ（＝下降トレンド終了）＋ 高値を更新 → 上昇トレンド転換確定
- ダブルボトム形成＋高値更新でも転換とみなせる

**重要な区別:**
- トレンド終了 ≠ トレンド転換（終了後、再度元方向に戻ることもある）
- 「高値切り下げだけ」「安値切り上げだけ」はトレンド終了に過ぎない

---

### ステップ5: 4つの鉄板エントリーパターン判定

収集した情報をもとに、以下の4パターンのどれに当てはまるかを判定してください。

---

## パターン① 4時間足レベルの押し目・戻り目

**最も頻出・最優先でマスターすべき鉄板パターン（全トレードの8割以上）**

**大きな流れ:** 4時間足のトレンド方向
**小さな流れ:** 1時間足

**エントリー条件:**
- 4時間足が上昇トレンド中 → 「押し目買い」を狙う
  - 4時間足レベルの高値切り上げポイント（安値切り上げを形成してくる局面）で
  - 1時間足が**下降トレンドから上昇トレンドに転換した瞬間**にロングエントリー
- 4時間足が下降トレンド中 → 「戻り売り」を狙う
  - 4時間足レベルの高値切り下げポイント（戻り目を形成してくる局面）で
  - 1時間足が**上昇トレンドから下降トレンドに転換した瞬間**にショートエントリー

**エントリーの具体的なタイミング（売りの例）:**
1. 4時間足: 下降トレンド継続中（赤80SMAが上からレートを抑えている）
2. 4時間足: 高値切り下げポイントを形成してこようかという局面
3. 1時間足: 上昇トレンド中（まだ売るタイミングではない）← ここでは売らない
4. 1時間足: **高値を切り下げ安値を更新 → 下降トレンド転換** ← ここで売りエントリー
5. このとき、既存のロング保有者の決済売り＋損切り売りも重なり、レートが伸びやすい

**追加要素（必ず確認）:**
- 要素①: 日足の方向とも合致しているか？（日足が反対方向ならエントリーしない）
- 要素②: 直上（買いの場合）or 直下（売りの場合）に反対勢力（レジスタンス/サポート・上位足MA）がないか？

---

## パターン② 1時間足レベルの押し目・戻り目

**パターン①の1段下の時間軸版（考え方は全く同じ）**

**大きな流れ:** 1時間足のトレンド方向
**小さな流れ:** 15分足

**エントリー条件:**
- 1時間足が上昇トレンド中 → 「押し目買い」を狙う
  - 1時間足レベルの安値切り上げポイントで
  - **15分足が上昇トレンドに転換した瞬間**にロングエントリー
- 1時間足が下降トレンド中 → 「戻り売り」を狙う
  - 1時間足レベルの高値切り下げポイントで
  - **15分足が下降トレンドに転換した瞬間**にショートエントリー

**概念:** 1時間足MAの一波（15分足MAの動き）を取りにいくトレード

**追加要素:**
- 要素①: 4時間足・日足の方向とも合致しているか？
- 要素②: 直上/直下に反対勢力がないか？

> **GOLD (XAU/USD) の注意点:** ゴールドは経済指標・地政学リスク・米ドル指数（DXY）に
> 大きく左右されるため、MA反応が通貨ペアより荒くなる場合がある。
> ボラティリティが高く値幅は大きいが、ダマシも多いので要素②（反対勢力確認）を特に厳格に行う。

---

## パターン③ 日足レベルの押し目・戻り目

**最も値幅が大きい・最も伸びやすいパターン**

**大きな流れ:** 日足のトレンド方向
**小さな流れ:** 4時間足（さらにその中の1時間足でエントリー）

**エントリー条件:**
- 日足が上昇トレンド中 → 「押し目買い」を狙う
  - 日足レベルの安値切り上げポイント（押し目）形成中に
  - 4時間足が**上昇トレンドに転換してくる局面**を確認
  - さらに下位足（1時間足）でエントリー精度を高める
- 日足が下降トレンド中 → 「戻り売り」を狙う
  - 日足レベルの高値切り下げポイント（戻り目）形成中に
  - 4時間足が**下降トレンドに転換してくる局面**でエントリー

**このパターンで獲れる波:**
- 日足レベルの波の中に、4時間足レベルの第1波・第3波が出てくる
- 場合によっては両方の波を獲ることができる

**エントリーのタイミング（日足押し目買いの例）:**
1. 日足: 上昇トレンド継続中
2. 4時間足: 下降トレンド中（＝日足の押し目形成中）← まだ待つ
3. 4時間足: 安値切り上げ＋高値更新 → 上昇トレンド転換 ← ここが「合致する瞬間」
4. 1時間足: さらに下位足で転換を確認してエントリー（より早いエントリー）

**注意:**
- 4時間足のトレンド転換ポイントでエントリーするのはデイトレとしては「遅すぎる」
- 4時間足高値切り下げポイント（A）の中の1時間足下降トレンド転換でエントリーする

---

## パターン④ 4時間足レベルのトレンド転換

**今まで続いていたトレンドが転換する局面をその根元から獲りにいくパターン**

パターン①〜③（押し目・戻り目）とは異なり、**新しいトレンドの発生点**を狙う。

### パターン④には3つのサブパターンがある

#### ④-A: 日足のトレンドに合致した4時間足レベルのトレンド転換
= 「日足レベルの押し目・戻り目（パターン③）」と同義

- 3サブパターン中で**最も値幅が期待できる**
- 日足方向と同じ方向に4時間足がトレンド転換するのだから当然伸びやすい

**エントリー:**
- まさにトレンド転換する局面でのエントリー、または転換後の初押しでエントリー

#### ④-B: 日足のトレンドに逆行した4時間足レベルのトレンド転換
= 「日足の押し目・戻り目形成のための起点」

- 日足が下降トレンド中 → 戻り目形成のための上昇の起点で出てくる
- 日足が上昇トレンド中 → 押し目形成のための下落の起点で出てくる
- 日足の大きな流れに逆らうため、値幅は④-Aより少なくなる傾向
- このトレンド転換の後は必ず「日足の方向に戻っていく」

**注意:** このサブパターンは日足方向に逆らうため優位性はやや低い

#### ④-C: 大きなレンジ相場での4時間足レベルのトレンド転換

- 日足のトレンドとは関係なく、強いサポート・レジスタンスラインによって起こる
- 上値をレジスタンスで押さえられ、4時間足レベルで高値の切り下げが起こる
- 下値をサポートで支えられ、4時間足レベルで安値の切り上げが起こる
- 値幅はレンジ幅によって変わる

---

### ステップ6: エントリー前に必ず確認する2要素

どのパターンでも、エントリー前に以下を必ず確認する：

**要素①: より上位足（日足）の方向と合致しているか？**

> いくら4時間足の押し目買いポイントでも、日足が「下降トレンド中・戻り売り局面」であれば、日足レベルのトレーダーが売ってくるためレートは伸びにくい。日足トレーダーと下位足トレーダーの思惑が一致している局面でのみトレードする。

**要素②: 直上（買い）or 直下（売り）に反対勢力がないか？**

> レジスタンスライン・サポートライン・上位足のMA・トレンドラインなど、反対勢力が近くにあるとレートが伸びない。近すぎる場合はエントリーを見送り、その水準を抜けてから考える。

---

### ステップ7: 実際のトレード判断フロー

```
【STEP1】日足チャートを確認
  → 日足のトレンドを把握（ダウ理論＆移動平均線）
  → 上昇なら「目線を上（押し目買い待ち）」、下降なら「目線を下（戻り売り待ち）」
  → レンジ・グチャグチャなら様子見（チャートが整うまで触らない）

【STEP2】4時間足チャートを確認
  → 4時間足のトレンドを把握
  → 日足と方向が一致しているか確認
  → 不一致（戦争状態）ならトレードしない

【STEP3】ラインの強さと反対勢力を確認
  → 直上のレジスタンス、直下のサポートを確認
  → 反対勢力が近ければエントリー見送り

【STEP4】1時間足（or 15分足）を確認
  → 大きな流れに合致する方向に転換するポイントを探す
  → 転換確定の瞬間にエントリー

【STEP5】決済ラインの設定
  → 損切り: 直近の押し安値（買い）or 戻り高値（売り）の向こう側
  → 利食い: トレンドが進むにつれ決済ラインを移動（トレーリング）
```

## 重要な原則（マニュアルより）

1. **トレンドフォロー一本** — 逆張りは使わない
2. **大きな流れに逆らわない** — 上位足の方向に沿う
3. **戦争状態では手を出さない** — 日足vs4時間足の思惑が異なる局面は避ける
4. **監視通貨ペアは絞る** — 初心者・まだ勝てない人は1〜3通貨に絞る（GOLD含む）
5. **武器は一つを極める** — まずパターン①（4時間足の押し目・戻り目）だけを徹底マスター
6. **デスクトップPC必須** — スマホのみでのFXトレードは論外
7. **週末持ち越しなし** — 金曜深夜にポジションをクローズ
8. **GOLD (XAU/USD) 対応** — 監視対象に追加。ただし地政学リスク・米CPI等のイベント時は特に注意

## 用語集

| 用語 | 説明 |
|------|------|
| プライマリートレンド | 上位足が示す大きな相場の方向性。これに逆らえない |
| セカンダリートレンド | プライマリーに対して一時的に逆行する動き（必ずプライマリーに戻る） |
| 押し目 | 上昇トレンド中の一時的な下落（押し目買いのチャンス） |
| 戻り目 | 下降トレンド中の一時的な上昇（戻り売りのチャンス） |
| 押し安値 | 直近の上昇波を作った時の安値（これを割ると上昇トレンド終了） |
| 戻り高値 | 直近の下降波を作った時の高値（これを超えると下降トレンド終了） |
| 戦争状態 | 上位足と下位足の思惑が一致せず、相場がごちゃごちゃしている状態。触らない |
| MA収束 | 複数のMAが集まってきている = 上位足と下位足が拮抗中 |
| MA拡散 | MAが広がっている = 一方のトレンドが優位を確立している状態 |
| 大きな流れ | 日足＆4時間足のトレンド |
| 中くらいの流れ | 4時間足＆1時間足のトレンド |
| 小さな流れ | 1時間足＆15分足のトレンド |
