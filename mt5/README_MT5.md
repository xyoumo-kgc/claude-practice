# FX4Pattern_EA — MT5セットアップガイド

## 概要

「4つの鉄板エントリーパターン解説マニュアル」のパターン①  
**「4時間足レベルの押し目・戻り目」** を自動売買するMT5 EA（Expert Advisor）です。

---

## ファイル構成

```
mt5/
├── FX4Pattern_EA.mq5          # EAソースコード（MT5でコンパイル）
├── FX4Pattern_Backtest.set    # ストラテジーテスター推奨設定
└── README_MT5.md              # このファイル
```

---

## インストール手順

### 1. EAファイルの配置

```
MT5インストールフォルダ\
  MQL5\
    Experts\
      FX4Pattern_EA.mq5   ← ここにコピー
```

**パスの確認方法:** MT5 → ファイル → データフォルダを開く → MQL5 → Experts

### 2. コンパイル

1. MT5のナビゲーター画面で `FX4Pattern_EA.mq5` を右クリック
2. 「コンパイル」を選択
3. 「コンパイル完了」メッセージを確認

### 3. チャートへのアタッチ

1. **H1（1時間足）チャート** を開く（重要: 必ずH1）
2. ナビゲーターから `FX4Pattern_EA` をドラッグ＆ドロップ
3. 「自動売買を許可」にチェック
4. パラメータを設定してOK

---

## パラメータ設定ガイド

| パラメータ | 推奨値 | 説明 |
|-----------|--------|------|
| `InpRiskPercent` | 1.0 | 1トレードのリスク（残高の1%） |
| `InpRiskReward` | 2.0 | リスクリワード比（SL:TP = 1:2） |
| `InpSL_ATR_Mult` | 2.0 | SL幅 = ATR(14) × 2.0 |
| `InpMA_1H` | 20 | 1時間足MA（青線） |
| `InpMA_4H_Equiv` | 80 | 4時間足MA相当（赤線） |
| `InpMA_Day_Equiv` | 480 | 日足MA相当（黄線） |
| `InpSlopeWindow` | 5 | MAスロープ判定ウィンドウ |
| `InpUseDailyFilter` | true | 日足フィルター（推奨：ON） |
| `InpAllowBuy` | true | 買いを許可 |
| `InpAllowSell` | true | 売りを許可 |

## ストラテジーテスター（バックテスト）手順

### 基本設定

1. MT5 → 表示 → ストラテジーテスター（または `Ctrl+R`）
2. 以下を設定：

```
EA名     : FX4Pattern_EA
シンボル  : USDJPY（または EURUSD / GBPUSD / XAUUSD）
時間足    : H1  ← 必須
モデル    : 全ティック（精度最高）
期間      : 2020.01.01 ～ 2025.06.01（5年間）
初期資金  : 1,000,000 円
レバレッジ: 1:25
```

3. 「入力」タブ → `FX4Pattern_Backtest.set` を読み込む
4. 「スタート」をクリック

### 推奨チェック項目（結果確認）

| 指標 | 目安 |
|------|------|
| プロフィットファクター | 1.3 以上 |
| 勝率 | 40% 以上（RR1:2なら40%超で期待値プラス） |
| 最大ドローダウン | 20% 以下 |
| 月利プラス月数 | 全体の60%以上 |

## シグナルロジック詳細

### エントリー条件（買いの場合）

```
条件①: 4時間足MAが上向き（MA80のスロープ > 0）
        → 4時間足は上昇トレンド中

条件②: 1時間足MAが直前は下向き
        → 押し目形成中（一時的な逆行）

条件③: 1時間足MAが現在上向きに転換
        → 押し目終了・上昇再開のサイン

条件④: 日足MAが下向きでない（日足フィルターON時）
        → 上位足との「戦争状態」を回避
```

### 決済条件

```
SL: エントリー価格 - ATR(14) × InpSL_ATR_Mult
TP: エントリー価格 + SL幅 × InpRiskReward
```

---

## 注意事項

- **このEAはデモ口座でのテストを推奨します**
- スプレッドの広い時間帯（重要指標発表前後）は手動で停止推奨
- GOLDは他の通貨ペアよりボラティリティが高いため `InpSL_ATR_Mult` を 2.5〜3.0 に上げることを検討
- EAはあくまでパターン①の機械的な近似です。実際のトレードでは目視による相場判断が加わります
